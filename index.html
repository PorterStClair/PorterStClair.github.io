Projects

Interpreter:

I built an interpreter using Java for a simple custom programming language. This language implemented block scope and was capable of reading user input, simple math and relational operations, if/else statements, while loops, and comments.

Task Tracker:

I worked with a group of 3 other students to build a Task Tracker app using Scala. We utilized ScalaFX to make an easy to use GUI. Each created task includes a name and end data and are automatically sorted prioritizing tasks that needed to be done first.

OpenStreetMap Analysis:

I wrote a Javascript program that recursively scans a user specified JSON file obtained through OpenStreetMaps's Overpass API. This scan finds, lists, and counts every instance of a user specified tag in the location represented by the JSON file. An example use of this program is to find the top contributors to OpenStreetMap in any area.
